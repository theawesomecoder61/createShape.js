Usage

Setting up

Easiest: CDN

Paste the code into your page's <head>.

<script src="http://theawesomecoder61.github.io/createShape.js/js/createShape.js"></script>

Easy: Default Javascript

<script src="path/to/createShape.js"></script>

Test Driving

Creating a canvas

Put this in your script's onload function. 

              var circle = createShape({
                canvasQuery: "#cs-canvas",
                appendTo: "body",
                canvasWidth: 400,
                canvasHeight: 400,
                shape: "circle",
                x: 100,
                y: 100,
                radius: 40,
                fillColor: "blue"
              });

Using an existing canvas

Put this in your script's onload function. 

              var circle = createShape({
                canvasQuery: "#cs-canvas",
                shape: "circle",
                x: 100,
                y: 100,
                radius: 40,
                fillColor: "blue"
              });
